# OperaFix V115 - Dashboard Builder solo en Dashboard KAM

## Corrección incluida

- El bloque **Paneles personalizados** ahora se muestra únicamente en la pestaña **Dashboard KAM**.
- Al entrar a:
  - Empresas
  - Kanban
  - Agenda
  - Campañas / Excel
  - Ranking KAM
  - Reglas

  ya no se mostrará el dashboard builder arriba.

## Motivo

El dashboard builder corresponde al panel ejecutivo del administrador KAM, no debe ocupar espacio visual en las vistas operativas como Empresas o Kanban.
