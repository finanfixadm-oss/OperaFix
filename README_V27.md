# OperaFix v27 - Filtros por mandante

Cambios principales:

- En Registros de empresas, los filtros avanzados, búsqueda rápida, columnas visibles y orden de columnas quedan guardados por vista/mandante.
- Al cambiar entre FINANFIX, Mundo Previsional, Optimiza, Mis Registros y Todos, cada vista recupera su propia configuración.
- En Dashboard ejecutivo, los filtros avanzados y columnas quedan guardados por mandante seleccionado.
- Se mantiene la barra horizontal y el orden manual de columnas de la v26.

Para instalar:

```bash
git add .
git commit -m "v27 filtros por mandante"
git push
```
