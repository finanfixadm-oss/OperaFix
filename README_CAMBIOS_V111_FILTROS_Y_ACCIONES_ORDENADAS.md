# OperaFix V111 - Filtros profesionales y acciones ordenadas

## Cambios incluidos

- Se eliminó el botón duplicado **Nueva empresa** del encabezado superior.
- Se dejó **un solo botón Nueva empresa** dentro del bloque del listado de empresas.
- El botón **Exportar solo filtrado** quedó junto al listado de empresas.
- El bloque **Filtros comerciales** fue rediseñado para verse más profesional.
- Los filtros ahora funcionan como una sección más limpia y desplegable.
- Se agregó opción **+ Agregar filtro** para ir mostrando solo los filtros necesarios.
- Se pueden quitar filtros visibles desde chips/píldoras.
- Se mantiene **Buscar** siempre visible para acceso rápido.

## Objetivo visual

- Reducir ruido visual.
- Evitar acciones repetidas.
- Ordenar mejor la experiencia entre filtros y listado.
- Dar una apariencia más tipo CRM profesional.
