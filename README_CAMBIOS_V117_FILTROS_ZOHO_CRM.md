# OperaFix V117 - Filtros estilo Zoho CRM

## Cambios incluidos

- Rediseño visual de los filtros comerciales para que se vean más estilo Zoho CRM.
- El buscador queda arriba como filtro principal.
- Los filtros avanzados quedan en una sección ordenada por filas:
  - Campo
  - Criterio / valor
  - Acción
- Cada filtro se puede quitar individualmente.
- Selector para agregar nuevos filtros de forma progresiva.
- Mejor uso de bordes, fondos, espaciado y jerarquía visual.
- Diseño responsive para pantallas pequeñas.
