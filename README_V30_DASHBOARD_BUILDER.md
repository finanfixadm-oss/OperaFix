# OperaFix v30 - Dashboard Builder

Incluye constructor de paneles tipo Zoho:

- Crear panel desde Dashboard Ejecutivo.
- Seleccionar tipo: tabla dinámica, barras, circular o KPI.
- Seleccionar medida: cantidad, monto devolución, monto Finanfix, monto cliente.
- Seleccionar hasta 3 agrupaciones con todos los campos disponibles.
- Agregar criterios de filtro propios por panel.
- Vista previa antes de guardar.
- Editar y eliminar paneles.
- Guardado por mandante seleccionado en localStorage.

No requiere librerías nuevas.
