# OperaFix V113 - Dashboard Builder KAM estilo Zoho

## Cambios incluidos

- Nuevo bloque **Paneles personalizados** en el módulo KAM.
- Permite crear componentes de información usando las columnas comerciales existentes:
  - RUT
  - Razón Social
  - Mandante origen
  - Campaña
  - Nro Empleados
  - Monto Dev.
  - Rubro
  - Región
  - Contactos
  - Nombre
  - Cargo
  - Correo
  - Tel. contacto
  - Tel. central
  - LinkedIn
  - Estado
  - Próxima gestión
  - Probabilidad cierre
  - Prioridad
  - Score
  - KAM
- Tipos de visualización:
  - KPI
  - Tabla agrupada
  - Barras
  - Circular / dona
  - Lista resumen
- Medidas disponibles:
  - Cantidad de registros
  - Suma
  - Promedio
  - Máximo
  - Mínimo
- Agrupamientos por una o más columnas.
- Filtros propios por componente.
- Vista previa antes de guardar.
- Opciones por panel:
  - Editar
  - Clonar
  - Eliminar
  - Ver registros utilizados

## Nota

Esta versión agrega el builder en frontend sobre los datos ya cargados y filtrados del módulo KAM. 
Más adelante se puede persistir cada panel en base de datos para que queden guardados por usuario.
